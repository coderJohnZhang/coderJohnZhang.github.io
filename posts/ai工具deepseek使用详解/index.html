Deepseek是什么

中国 AGI 领域的潜力之星

在当下人工智能蓬勃发展的时代，DeepSeek 作为一家专注通用人工智能（AGI）的中国科技公司，正崭露头角。它将主要精力放在大模型的研发与应用上，以技术创新为核心驱动力。

在研发过程中，DeepSeek 投入大量资源进行算法优化和模型训练，力求提升模型的性能与智能化程度。其研发成果已在多个领域得到应用，为各行业带来了新的解决方案和发展机遇，比如在智能客服领域，大幅提升了服务效率和客户满意度。

DeepSeek 的发展不仅体现了中国在 AGI 领域的技术实力，也为行业发展注入了新的活力。

开源推理模型 DeepSeek-R1

DeepSeek-R1是其开源的推理模型，擅长处理复杂任务且可免费商用。

DeepSeek-R1 的独特之处在于其强大的任务处理能力，尤其擅长数学、代码和自然语言推理等复杂任务。这得益于它采用强化学习进行后训练，构建了智能训练场，能动态生成题目和实时验证解题过程，极大提升了推理能力 ，在数学、代码和推理任务上可与 OpenAI 的 O1 相媲美。

而且，DeepSeek-R1 完全开源，采用 MIT 许可协议，这意味着开发者可以自由使用、修改和分发，甚至用于商业用途，无需担心版权问题。这不仅降低了 AI 应用的开发门槛，也为开源社区的发展注入了强大动力。

目前，DeepSeek-R1 已被众多企业和平台接入，如百度智能云千帆平台、阿里云、腾讯云等，应用场景不断拓展。无论是智能聊天、代码编写还是复杂的数据分析，DeepSeek-R1 都能展现出卓越的性能。

Deepseek可以做什么

它既面向普通用户，也为开发者提供支持，涵盖多种应用场景：智能对话、文本生成、语义理解、计算推理、代码生成与补全。它支持联网搜索与深度思考模式，还具备文件上传功能，可扫描并读取各类文件及图片中的文字内容，为用户带来更加便捷、高效的使用体验。

文本生成

  • 文本创作 在文本创作方面，它涵盖了多种类型的内容写作，比如文章、故事、诗歌等文学创作；商业用途的营销文案、广告语；社交媒体上的推文、帖子等内容产出；还包括剧本创作和对话设计，能辅助影视、游戏等行业的内容制作。
  • 摘要与改写 这个方向主要针对已有文本进行处理。可以对长文本如论文、报告等进行摘要提取，提炼关键信息；也能简化复杂文本，让内容更易理解；另外还能实现多语言翻译以及本地化处理，打破语言障碍。
  • 结构化生成 结构化生成是指创建具有特定格式的文本，像生成日程安排表、菜谱等表格和列表形式的内容；在软件开发领域，还能用于生成代码注释以及撰写相关技术文档 。

这些应用方向展示了文本生成技术在不同场景下的广泛用途。

自然语言理解与分析

  • 语义分析 语义分析旨在理解文本的含义，包括语义解析，分析文本中词汇和句子的语义；情感分析用于分析评论、反馈等文本中的情感倾向；意图识别能够识别客服对话、用户查询等文本背后的真实意图；实体提取则是从文本中提取人名、地点、事件等关键实体信息。
  • 文本分类 文本分类是将文本划分到不同类别中，比如生成主题标签，像对新闻进行分类；还可以用于检测垃圾内容，过滤掉不良或无关的信息。
  • 知识推理 知识推理通过已有的知识进行逻辑推导，能够解答数学、常识推理等逻辑问题，还可以进行因果分析，找出不同事件之间的关联性 。

这三个领域相互关联，共同推动自然语言处理技术在理解、分类和推理文本信息方面的发展。

编程与代码相关

  • 代码生成 可以根据开发者描述的功能需求，生成 Python、Java、JavaScript 等多种编程语言的代码片段，还能实现代码自动补全，提高编码效率。
  • 代码理解与解释 能够解读复杂代码逻辑，对代码的功能、各模块间关系进行解释，方便开发者尤其是初学者理解代码。
  • 代码调试与优化 帮助分析代码中的错误，给出修复建议；同时针对代码性能瓶颈进行分析，提供优化思路和方法。
  • 技术文档生成 基于代码生成对应的 API 文档、使用说明等技术文档，便于团队协作和知识共享。

常规绘图

AI 助力下能实现如下丰富绘图功能：

  • SVG 矢量图绘制：AI 可辅助生成 SVG 矢量图，用于创建基础图形、图标、简单插图、流程图和组织架构图。借助 AI 的算法，能快速精确绘制规则或复杂的矢量图形，且放大缩小不失真，在网页设计、UI 设计中广泛应用。
  • Mermaid 图表绘制：AI 能依据 Mermaid 语法规则，快速生成流程图、时序图、类图等。例如，在软件开发领域，开发人员用 AI 辅助绘制类图和时序图，梳理代码逻辑与交互流程；在业务流程梳理中，通过 AI 生成的流程图能清晰展示各环节关系。
  • React 图表绘制：利用 AI 可便捷创建 React 图表，如折线图、柱状图等。在数据可视化场景中，AI 能对大量数据进行分析处理，选择合适的 React 图表类型呈现数据特征与趋势，帮助企业和研究人员更直观理解数据，做出决策。

AI 基础对话技巧

与 AI 对话就如同结交新朋友，掌握恰当的技巧能让交流更顺畅、更有成效。

  • 有效提问的五个黄金法则

    • 法则⼀：明确需求 ✖ 错误示例：“帮我写点东西” ✔ 正确姿势：“我需要⼀封求职邮件，应聘新媒体运营岗位，强调3年公众号运营经验”
    • 法则⼆：提供背景 ✖ 错误示例：“分析这个数据” ✔ 正确姿势：“这是⼀家奶茶店过去三个月的销售数据，请分析周末和工作日的销量差异（附CSV数据）”
    • 法则三：指定格式 ✖ 错误示例：“给几个营销方案” ✔ 正确姿势：“请用表格形式列出三种情⼈节咖啡店促销⽅案，包含成本预估和预期效果”
    • 法则四：控制⻓度 ✖ 错误示例：“详细说明” ✔ 正确姿势：“请用 200 字以内解释区块链技术，让完全不懂技术的老人能听懂”
    • 法则五：`及时纠正 当回答不满意时，可以说：
      1. “这个方案成本太高，请提供预算控制在 500 元以内的版本”
      2. “请用更正式的语⽓重写第二段”

  • 新手必学魔法指令

    1. /续写 ：当回答中断时⾃动继续⽣成
    2. /简化 ：将复杂内容转换成大白话
    3. /示例 ：要求展示实际案例（特别是写代码时）
    4. /步骤 ：让 AI 分步骤指导操作流程
    5. /检查 ：帮你发现文档中的错误

  • 场景演练： 输/步骤如何用手机拍摄美食照片”，观察分步指导； 输⼊“请解释量⼦计算”，然后“/简化”，对⽐前后差异；

    演示如下： 

    import os
import requests
from bs4 import BeautifulSoup
from urllib.parse import urljoin
from tqdm import tqdm
import ssl
from urllib3.poolmanager import PoolManager
from requests.adapters import HTTPAdapter

# 处理SSL证书错误
class SSLAdapter(HTTPAdapter):
    def init_poolmanager(self, *args, **kwargs):
        context = ssl.create_default_context()
        context.set_ciphers('DEFAULT@SECLEVEL=1')
        kwargs['ssl_context'] = context
        return super(SSLAdapter, self).init_poolmanager(*args, **kwargs)

# 下载图片函数
def download_image(url, folder):
    response = requests.get(url, stream=True)
    total_size = int(response.headers.get('content-length', 0))
    block_size = 1024  # 1 KB
    filename = os.path.join(folder, url.split('/')[-1])
    progress_bar = tqdm(total=total_size, unit='iB', unit_scale=True)

    with open(filename, 'wb') as file:
        for data in response.iter_content(block_size):
            progress_bar.update(len(data))
            file.write(data)
    progress_bar.close()

# 主函数
def download_images_from_webpage(url, folder):
    if not os.path.exists(folder):
        os.makedirs(folder)

    session = requests.Session()
    session.mount('https://', SSLAdapter())

    response = session.get(url)
    soup = BeautifulSoup(response.text, 'html.parser')

    for img in soup.find_all('img'):
        img_url = img.get('src')
        if img_url:
            img_url = urljoin(url, img_url)
            print(f"Downloading {img_url}")
            download_image(img_url, folder)

# 使用示例
if __name__ == "__main__":
    webpage_url = "https://example.com"  # 替换为你要下载图片的网页URL
    save_folder = "downloaded_images"  # 替换为你要保存图片的文件夹路径
    download_images_from_webpage(webpage_url, save_folder)
